# VECTOPLAN Trademark Guidelines

The VECTOPLAN name, VECTOPLAN logo, official VECTOPLAN editor name, VECTOPLAN Cloud name and related product names are protected identifiers of the VECTOPLAN project.

The software license for this repository does not grant permission to use VECTOPLAN trademarks except where allowed by applicable law or these guidelines.

## Allowed use

You may use the VECTOPLAN name to:

* refer to the original VECTOPLAN project
* describe compatibility with VECTOPLAN file formats
* state that your project is based on or derived from VECTOPLAN software, if that statement is accurate
* link to the official VECTOPLAN project

Examples:

```text
Compatible with VECTOPLAN files.
Built using VECTOPLAN VPLIB tools.
Forked from vectoplan-editor.
```

## Not allowed

You may not:

* present your fork as an official VECTOPLAN release
* use the VECTOPLAN name or logo as the main name or branding of your own product
* remove or obscure VECTOPLAN copyright, license or attribution notices
* use VECTOPLAN branding for a hosted, managed, white-label or competing platform
* imply endorsement, partnership or certification without written permission
* register domains, package names, app names or social media names that imply official VECTOPLAN ownership

## Forks and modified builds

Modified versions must be clearly marked as modified.

They must not use names such as:

```text
VECTOPLAN Pro
VECTOPLAN Cloud
VECTOPLAN Editor Official
Official VECTOPLAN Build
```

unless explicitly approved by VECTOPLAN.

Acceptable descriptive names include:

```text
MyProject, based on vectoplan-editor
Unofficial fork of vectoplan-editor
Custom internal build of vectoplan-editor
```

## Official builds

Only VECTOPLAN may publish official VECTOPLAN builds, installers, cloud services, app store releases or branded distributions.
